# 🕵🏻‍♂️Kimlik Komutları

## Kimsin

Sunucudaki üyenin discorda ne zaman kayıt olduğundan tut eğer kayıtlıysa ismini yaşını vb. bir çok konuda bilgilendirir.

{% code title="Kullanım:" %}
```scala
!kimsin [@kişi/id]
```
{% endcode %}
