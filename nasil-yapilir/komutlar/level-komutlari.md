# 🏆Level Komutları

## Top

Sunucunun Genel ve haftalık olmak üzere ilk 5 chat ve sesli sıralamasındaki üyelerin listesini görüntüleyin.

{% code title="Kullanım:" %}
```scala
!top
```
{% endcode %}
